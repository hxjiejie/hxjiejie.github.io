Cache 门面解析

Cache门面：

Cache门面继承Illuminate\Support\Facades

return 'cache' 其实，就是在容器中的实例名称。

以下三种方法对于生成门面是至关重要的：

  • __callStatic() PHP魔术方法
  • getFacadeRoot 获取实例
  • getFacadeAccessor 获取注册在容器中的实例名（不重写该方法会抛出一个异常）
